# Project template

This is just an exampleof the `README` file that is used by the template. Please update it to reflect the content of your project.
This template adds the following:

- A `devcontainer file` to be used in `vscode` like development enviroments.
- Templates for the creation of issues:
  - Bug request
  - Chore requests
  - Feature requests
  - Documentation requests
  - Dailys request (for keeping track of dayly meetings)
  - Task requests
  - Test request
